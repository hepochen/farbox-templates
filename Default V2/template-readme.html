关于默认模板的说明

这是FarBox上网站的默认模板，本版于2013-4-25第一次发布。它的基本功能如下:

  • 自动判断并兼容相册、博客
  • 页面宽度自适应，兼容移动设备，如iPad、iPhone
  • 相册功能，自动识别照片文件夹(文件夹若无图则会过滤掉)
  • 自动识别links.md、about.md为Links与About页面
  • 文章与照片页面，自动识别comment_js.md的第三方社交评论或者用FarBox的评论系统

修改网站描述与其它项目?

如果需要修改本模板的自定义项，需要在网站目录（比如FarBox/My Blog/）下放置一个名为site.md的文件，如下是一个范例:

head_bg: #ca2f33
cover: http://resource.farbox.com/image/cover.jpg
cover_author: Shutterhacks
cover_link: http://www.flickr.com/photos/shutterhacks/4474421855/

写作，是对生命的一种礼赞。
我们沉浸于生活的时候，会遗忘生活赋予我们的理想；我们停下脚步，用笔尖与思想同黑夜对话的时候。
我们就感觉到了自己。

格式说明:

head_bg: 顶部色块的颜色
cover: 顶部图片的地址
cover_author: 顶部图片的作者，鼠标悬浮的时候会显示。
cover_link: 题图的源网址，表明有版权署名的需要，并且会在网页的底部保留这个网址。

这里是写网站的描述，不易过长，会出现在顶部图片（题图）的文字中。
以上的每一项都不是必须的，按需修改。

文章自定义项目

同网站自定义项一样，支持head_bg, cover, cover_author；以及另外一个属性为foreword来修改题图上的文字。

head_bg: 顶部色块的颜色
cover: 顶部图片的地址
cover_author: 顶部图片的作者，鼠标悬浮的时候会显示。
cover_link: 题图的源网址
foreword: 这里的文字会替换题图上的文字

这里开始是文章的正文 .etc

如果文章的foreword必须要换行的，则要遵从一个更严谨些的规则（YAML）。

---
head_bg: 顶部色块的颜色
cover: 顶部图片的地址
cover_author: 顶部图片的作者，鼠标悬浮的时候会显示。
foreword: |
    保证前面的空格，第一行
    保证前面的空格，第二行
    不要忘记foreword:后面的 |
---

这里开始是文章的正文 .etc

注意事项：

1，如果使用YAML规则，请确保文章的第一行是以三个-+换行开始的，并且三个-+换行后才能开始写正文, 以及:是英文字符并且后面有空格。

2，具体的设置项按需即可，不需要全部设定的。
